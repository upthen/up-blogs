# 常用 VSCode 插件

## Markdown 插件

1. Markdown All in One

vscode 内置的 markdown 编辑器无法支持快捷键添加格式，可安装此插件支持
(纠结了一下，还是卸载了，这个插件覆盖了太多默认快捷键，改变了很多默认输入行为，让人使用很不习惯，还是回归最本质的手动输入体验吧)
