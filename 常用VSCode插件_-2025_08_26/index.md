# 常用 VSCode 插件

## Markdown 插件

1. Markdown All in One

vscode 内置的 markdown 编辑器无法支持快捷键添加格式，可安装此插件支持

2. Auto Close Tag
   自动闭合标签
3. Auto Rename Tag
   自动重命名标签
4. Better Comments
   优化注释
5. Code Spell Checker
   英文拼写检查
6. Draw.io Integration
   draw.io 插件
7. Error Lens
   行内展示错误信息
8. Eslint
   eslint
9. FnMap
   函数地图
10. i18n Ally
    国际化插件
11. Import Cost
    导入包大小
12. Live Server
    静态服务器
13. prettier
    格式化代码
14. Version Lens
    代码插件
15. GitLens
    行内查看 git 提交记录
