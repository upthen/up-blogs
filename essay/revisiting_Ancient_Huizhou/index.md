# 再游"古徽州" <sup><a href="#影集" op:60 hover:op90 title="点这里，直接看旅途风景"><div class="i-ic-outline-photo-camera-back inline-block cursor-pointer" /></a></sup>

> 二零二五年十月于武汉江夏

结束了八天国庆长假，生活回归平淡，休息一日后，终于得暇来动笔写点什么，毕竟度过了一个非常轻松惬意的假期，终是不舍，不愿辜负，不擅写作，暂随心所欲的写吧。

原定国庆假期是要去北方的，第一选择是呼伦贝尔，第二选择是青甘大环线。但终因国庆假期机票涨价太离谱而不得不放弃。最后想着寻一个僻静的古村落安静的住几天，躲躲清闲，不需负盛名，不必多惊艳，只要够安静。

在小红书和抖音上看了不少古镇古村落的攻略，还是觉得黄山给人的感觉最好。大约是过往在黄山的旅游经历给我的判断添加了一层滤镜的缘故吧。

约四年前我和夕月便一起去过黄山一次，那也是在一个中秋佳节。尤记得当年的四天行程，我们游览了婺源篁岭，黟县宏村，也曾夜宿黄山之颠，一览黄山绝美的日出日落。美好的旅行体验让我们多年以后仍经常回味，也曾约定以后要再游黄山，毕竟在黄山的各处还隐藏着许多世外桃源。

决定了，这次旅行我们要再游“黄山”。

上面说的黄山，指的是黄山市。上世纪八十年代，当地政府为响应国家号召，利用好黄山，发展旅游产业，故而将地名改名为“黄山市”，而改名前的地名则更加为人所熟知—— “徽州”。

“一生痴绝处，无梦到徽州”。我们对徽州的印象，也许大多来自这句诗吧。这首诗是明代著名戏曲家汤显祖所作，完整内容是

<div  text-center border-1px border-solid border-aux1 rounded p-5>
欲识金银气，多从黄白游。<br/>
一生痴绝处，无梦到徽州。
</div>

诗人说，世人想要沾染金银之气，大多都是往黄山白岳山（今齐云山，在休宁县境内，道教发源地）一带游览。世人都执着于去那多商、多金的徽州，我就连做梦都不想去到那里。

当时徽州非常富裕，令世人趋之若鹜，都想去那里找点门路、混个机会，但诗人却明显不屑于此。这首诗语言浅近，态度鲜明，显示出诗人藐视庸俗的功名利禄，不愿折腰攀附权贵的高洁人格。

古徽州下辖 歙县，黟县，祁门，休宁，绩溪，婺源六县，地理环境表现出山多地少的特征，有“七山一水一分田，一分道路和庄园”之说，因山多田少，徽州地区长期处于粮食短缺状态，因而徽州人不得不背井离乡，从事商道。因此徽州又有“前世不修，生在徽州。十三四岁，往外一丢”的说法。很多徽州人从小就被迫背井离乡外出学习经商。徽商鼎盛时，曾有“无徽不成镇”的说法，曾垄断多个行业，商业版图遍布全国。清末著名的“红顶商人”胡雪岩就是徽商的典型代表。

今天的徽州，虽不复当年徽商鼎盛之光景，但其千百年来积累的丰富历史遗产以及其秀美的自然风光仍然让徽州熠熠生辉，令人向往。

这次我们去的地方是黟县的“卢村”。徽州较有名的古村落大多在黟县，最出名的要数宏村和西递，除此之外，还有卢村，碧山村，南屏村，屏山村等等。

我们原定计划是去“碧山村”，后来因住宿问题不得不另寻他处，选中卢村亦是阴差阳错，却也歪打正着。

卢村与宏村相邻，沿省道从宏村行驶不过三公里即可到卢村。从黄山北站直达卢村的公共交通是没有的，景区直达车的终点只到宏村，到宏村后需打车前往卢村。当然，如果在卢村订好了酒店或民宿，则可以让民宿老板来接，只要不是特别忙碌无法脱身，他们都是愿意接送客人的。徽州人很友善朴实，对待游客很热情，很真诚。

我们十月四号从武汉出发，坐高铁直达黄山北站。北站出站后往右手边走约几百米就是客运站，客运站有很多从北站到各景区的旅游专线。到宏村的专线票价30元每人，车程约100分钟。国庆期间自驾游颇多，宏村卢村一线的省道较为拥堵，我们到达宏村后，打电话联系民宿老板，民宿老板骑电动车来接的我们，在村内住宿门票享5折优惠。

卢村是一个开放性的景区，只在行人常走的一个大路口设了一道门禁，其余入口皆是开放的，只留一个村民看守。这里游客不多，每个入口由一个村民循例检查门票。 相比宏村，许是因为这里不设门禁，倒让人没有景区特有的束缚感，心情上也便轻松许多。村子里有一条小河（或可说小溪）横贯东西，蜿蜒流出。大部分民居皆依水而建，村民在河里浣衣，孩童在河边嬉戏。青山绿水，白墙黛瓦，流水潺潺，岁月静好。

村子商业化程度不高，只有必要的几个饭店和商店，不过民宿倒挺多的，似乎家家户户都预留了客房来充当民宿，这大概是卢村的另一种热情吧。

卢村始建于南唐末年 （约10世纪），距今已有超过千年的历史。据《黟县雉山卢氏宗谱》记载，卢氏先祖源出姜姓，因战乱采食于芦地，改姓为“卢”。汉初燕王卢绾后裔迁至安徽宣城，其后代卢振迁至太平县葛村（今黄山区郭村）。南唐时，卢振第三子卢玄因喜爱黟北地区“山川秀丽、土沃泉甘”，从葛村迁至雉山，成为卢村建村始祖，并以姓氏命名村落为“卢村”。卢氏家族以家训族规治家，强调“天下为公，民心为大”，通过私塾教育、楹联雕刻等方式传承家训，维系宗族凝聚力。历史上文风昌盛，名人辈出，如北宋进士卢臣忠（被宋高宗赠谏议大夫）、南宋左丞相卢麟等。卢村以其木雕闻名，清代道光年间，卢氏三十三代传人卢邦燮 （卢百万，官至四品朝议大夫）经商致富后，修建了以志诚堂木雕楼为代表的建筑群，木雕由工匠耗时20年完成，被誉为“徽州木雕第一楼”，成为徽派艺术的典范。

我们居住的民宿叫“卢村木雕楼农家饭庄”，是一家沿溪而建的朴素农家民宿。出了民宿，沿溪而下可直达村子出口，沿溪而上，顺着一片菜地向上走，不久便可达“卢村观景台”。卢村观景台是村子后方一处地势略高的平地，在这里可以俯瞰卢村全貌。傍晚和清晨从这里观景是最美的，近处的稻田，中间的黛瓦白墙，远处的群山，天边的朝霞夕阳，和着山间的微风，脚下的小桥流水，人间风景，莫过如此。

卢村的日出我们是十月六日早上去看的，日出的时间大约是6点钟，可以参考手机天气预报上的日出时间。六点钟是黄山的日出时间，卢村的日出要略晚一些，大约晚半个小时。我们五点半起床，五点四十五分出门，约五点五十五分到达观景台的。到达时，已有不少游客在观景台守候了。虽然太阳没有按照预定的时间出现，但等待的过程却并不枯燥，似乎相较于得到，期待的感觉反而更加美妙。晨曦之下的天空，光线柔美，远处的村庄云雾缭绕，别有一番意境，反而日出的过程倒显得平淡了。

而十月五日我们则去的是塔川和木坑竹海。十月四日我们起的颇早，而头天晚上又睡的晚，晚上因为收拾行李，而早上则要赶火车。对于习惯晚睡晚起的当代年轻人来说，连续两个早上早起是万万无法承受的，因此我们十月五日决定睡懒觉到中午，因此看日出只能推到十月六日早上了，而仅存的半天时间，也只够我们游览塔川和木坑竹海了。所幸我们所求不多，惟闲适尔，因此这样的行程却也是极好的。

上午去塔川，下午去木坑竹海。

塔川也是一个开放性村落，各个路口由村民看守。国庆期间在宏村、卢村等任意一个景点有购票记录，到塔川购票可享半价。

塔川是徽州黟县宏村镇的一个徽派古村落，以其被誉为"中国四大秋色之一"的绝美秋景而闻名。村内古民居依山而建，层层叠叠，形如宝塔，又有溪流穿村而过，故得名"塔川"。塔川最负盛名的是其秋季景观，与四川九寨沟、北京香山、新疆喀纳斯并称为"中国四大最美秋色"。其秋色的灵魂树种是乌桕和枫香。乌桕树叶会由绿变黄，再变红甚至变紫，色彩斑斓且富于变化；枫香则点缀其间，与徽派建筑的白墙黛瓦、马头墙相互映衬，在晨雾和炊烟中构成一幅生动的"秋日民居画卷"。

最佳观赏期通常在10月下旬至11月下旬，其中11月上中旬色彩层次最为丰富，是观赏的黄金季节。清晨（6:30-7:30）是拍摄晨雾、炊烟与红叶交融美景的最佳时间。我们去的时候树叶还没有变色，又赶上气温骤升，俨然是夏天的光景，传闻中那久负盛名的绝美秋景终是没有眼福。不过世事本就无常，不可能尽善尽美，我们倒也不挂怀，以后还会再来徽州也未可知。

塔川和木坑竹海相邻，由塔川沿省道一路向上游骑行不久，经过一个隧道便到了木坑竹海景区。木坑竹海有2个大门，南门和西门，可以由西门或南门上山，坐滑索下山，或着从南门游览木坑竹海一圈，依旧从南门的另一条路下山。推荐从南门上，从南门下，只有这条路线方能饱览竹海的绝美风光。

满山的竹子，郁郁葱葱。竹皆高洁，特立独行，根根笔直。过去也曾游览过不少名山，泰山、黄山、庐山、大理苍山…，但是，却从未有任何一个名山的树林给我以竹海这种感受，难以言说，如果真要描述，我觉得竹海的林子有一种“君子”或者说“士”的气质具像化的感觉。山风轻拂，竹海激荡，竹波汹涌，每一棵竹子皆独立，不攀附，虽随风左右摇摆，但其躯干始终保持笔直，不弯腰曲背，不怪乎竹子有四君子之美称。
￼
值得注意的是，李安导演、周润发杨紫琼主演的奥斯卡获奖电影《卧虎藏龙》就曾在宏村和木坑竹海取景，电影里“竹林斗剑”这场经典打戏就是在木坑村拍摄，《卧虎藏龙》能取得如此成就，宏村和木坑竹海这两个取景地也是功不可没。

在卢村待了2天，体验了徽州恬淡闲适的乡村生活，尽看了卢村的日出日落，畅游了《卧虎藏龙》中那片清幽风凛的竹山竹海，也吃到了徽州的臭鳜鱼，山笋，此行无憾矣。

十月六日上午，看完日出后即出发往徽州古城去了。民宿老板骑一辆电动三轮车送我们，我和夕月各拿一个小马扎坐在三轮车后车厢。三轮车在乡村小路上行驶时，我们仿佛梦回童年，每逢出行，我们也是如此坐到家里的三轮车后车厢，摇摇晃晃，走走停停，无忧无虑，平静而幸福。

我们从宏村拼了一辆出租车直达了徽州古城门口，车费120元。徽州古城既是一个景区，同时也是一个开放的小城，景点和民居混合分布在古城各处。古城由一圈古城墙围绕，古城里有“徽州府衙”、“许国石坊”、“斗山街”、“徽州历史博物馆”、“蛤蟆井”等景点。古城里相比古村落差异较大，虽然气温较高，但古村落因为近山近水，也不热，而古城则不同，行走在古城的大街小巷，有种在蒸笼中的感觉，徽州的窄巷在古村落中因为气是流动的，有凉爽的巷子风，而古城附近既没有群山，也没有流水穿过城中，反而古城外层围绕着新城，所以非常热，热到人只想逃离。

徽州府衙这些收费景点是没有去的，倒是“徽州历史博物馆”值得一逛。游览过许多城市，近年才明白一件事来，想了解一个地区一个城市，那么她的博物馆是必须要逛的。徽州历史博物馆内详细记录了徽州的历史，文化，民俗，建筑，饮食，徽商等方方面面的情况，逛完博物馆才算对徽州有一个比较全面和相对深入的认识，那时我才知道原来黄山市就是“古徽州”，也深感遗憾，徽州从此变成一个文化符号，而作为一个城市，她消失在历史中了。

十月六日晚上，我们在徽州古城中闲逛了少许时间，早早就回去休息了。古城晚上有鱼灯表演，但是游人实在太多，让人没有一丝观赏的兴致（当然，一部分原因是表演被游人围的水泄不通，也很难挤进去了）。第二天一大早我们便计划离开古城，往火车站去了。

遗憾的是，徽州城外的“新安碑园”终究是错过了，新安碑园中收藏了《余清斋帖》和《清鉴堂帖》这两套珍贵的明代发帖刻石。十月七日一大早我便往新安碑园去了，岂料去的太早，碑园尚未开门。而我们的行程决定了我不能久等，最后也只能留下一个小小遗憾。

坐上从徽州古城去往黄山北站的游1号线旅游公交车，此次的古徽州之旅也接近尾声。
我们到达车站后，寄存了行李，又往“西溪南”古村落去了。大概游玩了三个小时之后又重新回到了黄山北站，既而乘高铁回武汉了。相比卢村、宏村、黄山等给我的惊艳，古徽州和西溪南的旅程只能算平淡了。

End

游记最后，再来回顾一下这次国庆8天长假，很圆满，很充实，很幸福。十月一日上午和夕月一起开车回家，车多路堵，一个菜鸟司机遇上这种节假日的路况，竟然开的还挺顺利的，也是让我开车更加自信了。十月二日是一个圆满的团圆日，我们一家子和夕月自不必说，姐姐、姐夫、小外甥女、叔叔，堂姐，堂姐夫也都在当天上午从武汉回来，一起吃到一顿团圆午饭。不同的是，我也贡献了两个“拿手好菜” —— 红烧武昌鱼和香辣基围虾，获得一致肯定，相信以后厨艺会越来越好。十月三日又启程从家里开车回到武汉，因为是假期中间，高速就不是太堵了，开车开的还比较舒服。当天晚上收拾了出行的行李，第二天一早就启程出发前往“徽州”了。十月四日到十月七日，在“徽州”度过了一段幸福满足的旅程。十月八日，我和夕月一起陪她妈妈我未来岳母到华农游览，她跟同学借了一辆电动车，载着妈妈，而我则扫一辆共享单车，在后面跟着（动力驱动的还是比双脚驱动的车要快的多，在后面给我累的够呛）。在毕业典礼时带爸爸妈妈到学校里来，看看她每天学习生活的地方（武汉华农和三亚崖州湾）是她的心愿，这次终于圆了部分心愿了，真的很开心。下午，她给妈妈约了一辆顺风车，送妈妈回老家了，然后我们也回到武汉的小家。晚间，平静幸福的国庆假期在一场酣梦中结束。再天亮，又是新的一天了…
