# 我的第一台 Macbook 

> 2026年1月10日于武汉江夏

买一台 Mac 电脑是多年以来的一个小梦想，说梦想有点太大了，心心念念可能更贴切一些。

我曾经在 2018 年的时候，买了一台二手的 Mac mini，具体什么版本不记得了。

我记得当时还是研究生二年级，在潘老师的公司实习，公司在金融港，我和小米在当代国际花园合租房子。那时在京东拍拍上看到二手的 Mac mini 很便宜，大概花了不到3000块买了一台。第一次使用Mac那种欣喜难以言表，虽然是一台不知道落后多少代的二手Mac，但那种丝滑的使用体验真的不是 windows 电脑能比的。使用了3天后，还是退掉了。

理智告诉我，需要的并不是一台Mac mini，尽管那个方方正正的小盒子外观是如此优雅、内在又是如此强大。当时的我更需要的是一台windows 笔记本，兼具轻薄和性能。我后来买了一台华硕的灵耀笔记本，算是我当年买得起的最好的本子了。那台电脑陪我一起完成了我的毕业论文，然后，我毕业了。我这个人对于新的东西总抱有一种过分怜惜的态度，因此，任何物件我总是习惯买新用旧。当我有一台新电脑时，我才舍得可劲折腾就电脑。所以，我毕业时，我的旧电脑才刚开始发挥其效力，而我那台新买的华硕笔记本反而用的很少。所以，直到今天，它其实成色还不错，换过一次电池，加过一根内存条。不过不跑程序、搞开发，编辑文档、做ppt还是非常够用的。
