# up-blogs

一套 **极简** 风格的自定义 `vitepress` 博客模板，遵循让阅读回归本源的宗旨和极简的理念。

首先是自用，目前正在开发阶段，等达到一定完成度，讲抽离成一套主题模板进行开源。
