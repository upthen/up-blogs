# blog-archive

归档博客文章到 `/archive/` 目录。

## 使用

```bash
/blog-archive
```

## 工作流程

1. 扫描 `coding/` 和 `essay/` 目录下的所有文章
2. 弹出多选框，展示待归档文章列表
3. 选择需要归档的文章（支持多选）
4. 自动移动到 `/archive/` 目录

## 实现细节

**扫描规则：**
- 扫描目标：`coding/` 和 `essay/` 目录
- 排除文件：`index.md`
- 提取信息：文章标题（frontmatter.title）和目录名

**用户交互（分批询问）：**
- 由于文章数量较多（可能超过 30 篇），采用分批询问策略
- 每批显示 4 篇文章，使用 `AskUserQuestion` 的 `multiSelect: true`
- 循环询问所有批次，收集用户选择
- 格式：标签显示文章标题，描述显示完整路径

**归档操作：**
- 汇总所有批次的选择结果
- 使用 `mv` 命令将选中的目录移动到 `/archive/`
- 保留原目录结构和文件名

**输出结果：**
- 显示归档成功的文章列表
- 统计归档数量和失败情况（如有）
